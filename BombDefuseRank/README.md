# Defuse The Bomb

| Tecnologia | Versão |
|---|---|
| PHP | 7.4 - 8.1 |
| Laravel | 8 |

## Como rodar?

- Instalando Dependências:
```
  composer install
```

- Roda Migrations e Seeders:
```
  php artisan migrate:fresh --seed
```
